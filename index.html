Welcome to The Phantom Studios official website!

Hello guys! This is the official website of The Phantom Studios. I have been dedicating a long time to water development since 2020 when I started using unity. I focused on High Definition and Universal Render Pipelines, but i have the intention of making a package for built-in render pipeline.

AURW project

The Another Ultra Realistic Water (A.U.R.W.) is a big project, this is a package that helps unity developers with an already made water, depending of the version, you have different types of water.
